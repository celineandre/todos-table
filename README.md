# Todos Table JS

Le but est d'afficher dynamiquement les données JSON d'une liste de tâches dans un tableau HTML accessible avec uniquement du JavaScript Vanilla (exemple index.html).

Fonctionnalités :

* Trie ascendant ou descendant des colonnes du tableau
* Pagination
